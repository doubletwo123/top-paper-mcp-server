# Top Paper MCP Server

Use this server when you want research-grade access to academic papers from a Codex-compatible MCP client.

## Good fits
- Search arXiv by topic, date, category, or author
- Search papers from top conferences (CVPR, ICCV, NeurIPS, ICLR, ICML, AAAI, IJCAI, ECCV)
- Download and read full papers
- Traverse citation graphs
- Run semantic search over locally downloaded papers

## Install/runtime
- Package: `top-paper-mcp-server`
- Runtime: `uvx top-paper-mcp-server`
- MCP config: `../.mcp.json`
